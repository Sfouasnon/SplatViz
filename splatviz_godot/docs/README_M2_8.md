# SplatViz Godot M2.8

M2.8 adds the first practical analysis pass on top of the M2.7 UI:

- Focus readout now includes a color legend.
- Focus inspector includes camera distance, DoF, sensor/capture, optimal sharpness zone, and resolution viability.
- Splat Viability mode includes all-frustum and selected-only modes.
- Splat frustum opacity is reduced.
- Premium 36 marks candidate redundant same-tier neighbor cameras with small red markers.
- Inspector explains portrait vs landscape roll rationale per selected camera.
- Compare Layouts panel now includes camera-count, resolution, roll, and redundancy policy notes.

Predictions remain hypotheses. Msplat is a local smoke test. Production conclusions require gsplat validation.
