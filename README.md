# SplatViz

SplatViz is a Godot-based planning and validation tool for multi-camera Gaussian splat capture.

Current prototype goals:

- visualize camera rigs, frustums, focus zones, and splat viability
- export synthetic camera stills
- generate Msplat-compatible datasets
- inspect resulting .ply splats in the same stage coordinate frame
- iterate toward camera-truth validation for production gsplat workflows

## Current stack

- Godot 4.6.x .NET / Mono build
- GDScript prototype UI and viewport
- Msplat training via local CLI
- COLMAP-format sparse bridge generated by SplatViz
